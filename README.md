<<prerequisites and developer installations >> 
--g++ binary 
--SDL3 library 

if its windows
   install mingw and install the sdl3 package manually 
   or use the msys2 pacman to install the sdl3 
   then link the cpp file to the SDL dll and execute 
   notpad or cursor as IDE 





if its arch or debian in wsl2 
   enable wsl2 
   wslg 
   install gcc and g++ 
   install sdl3 
   compile and link 
   execute 
   vim / nvim or emacs or notepad.exe 




if its linux 
   just install the sdl3 package 
   execute the code by linking 
   vim nvim or emacs
